# Product Overview

A personal portfolio landing page for a GitHub ML/AI professional — designed to make a strong first impression on visitors arriving from a GitHub profile, CV, or shared link.

## Core Capabilities

- **Bio section**: Brief personal background and introduction
- **Project gallery**: Curated list of public GitHub repos with thumbnails, descriptions and direct links
- **Static delivery**: Zero JavaScript by default; fast, accessible, no client-side overhead
- **Live data**: Repos fetched from the GitHub API at build time so the gallery stays current

## Target Use Cases

- A recruiter or collaborator lands from a GitHub profile and wants a quick overview
- A CV/résumé link recipient wants to see work samples without navigating GitHub
- The owner shares a clean link instead of a raw GitHub URL

## Value Proposition

Simple, fast, and automatically up-to-date. No CMS to maintain — the GitHub API is the content layer. The site redeploys on every push, so the portfolio reflects the owner's current work at all times.
